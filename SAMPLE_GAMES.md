# サンプルゲーム推奨リスト

SteamDeck Mobile アプリで動作テストに使用できる無料Windowsゲームのリストです。

## 推奨テストゲーム

### 1. Cave Story (洞窟物語)
- **ライセンス**: フリーウェア
- **サイズ**: ~2MB
- **ジャンル**: 2D アクション・プラットフォーマー
- **要件**: DirectX 7以降、Windows 98+
- **ダウンロード**: https://www.cavestory.org/
- **推奨理由**:
  - 軽量で動作確認に最適
  - 純粋なネイティブWin32アプリ
  - コントローラー対応
  - Wine/Protonでの動作実績あり

### 2. SuperTux
- **ライセンス**: GPL（オープンソース）
- **サイズ**: ~150MB
- **ジャンル**: 2D プラットフォーマー（スーパーマリオ風）
- **要件**: OpenGL 2.0以降
- **ダウンロード**: https://www.supertux.org/
- **推奨理由**:
  - オープンソース
  - クロスプラットフォーム（Windows版あり）
  - コントローラー対応
  - 3D加速のテストに最適

### 3. OpenTTD (Transport Tycoon Deluxe)
- **ライセンス**: GPL（オープンソース）
- **サイズ**: ~30MB
- **ジャンル**: 経営シミュレーション
- **要件**: Windows XP+
- **ダウンロード**: https://www.openttd.org/
- **推奨理由**:
  - 軽量で安定
  - 2Dグラフィックス（Wine互換性高い）
  - マウス・キーボード操作

### 4. Spelunky Classic
- **ライセンス**: フリーウェア
- **サイズ**: ~20MB
- **ジャンル**: ローグライク・プラットフォーマー
- **要件**: DirectX 8.1以降
- **ダウンロード**: https://spelunkyworld.com/original.html
- **推奨理由**:
  - コントローラー対応
  - 人気のインディーゲーム
  - リプレイ性が高い

### 5. FreeDoom
- **ライセンス**: BSD（オープンソース）
- **サイズ**: ~30MB
- **ジャンル**: FPS（DOOMエンジン）
- **要件**: DOOMエンジン（GZDoom推奨）
- **ダウンロード**: https://freedoom.github.io/
- **推奨理由**:
  - 3D FPSのテストに最適
  - OpenGL レンダリングテスト
  - キーボード＋マウス操作

## テスト項目

各ゲームで以下をテストすることを推奨:

### 基本動作
- [ ] ゲームが起動する
- [ ] メインメニューが表示される
- [ ] ゲームプレイが正常に動作する
- [ ] ゲームを終了できる

### パフォーマンス
- [ ] FPS が 30fps 以上維持される
- [ ] 音声が正常に再生される
- [ ] ロード時間が許容範囲内

### 入力
- [ ] コントローラーが認識される
- [ ] ボタン入力が正しく反映される
- [ ] アナログスティックが機能する
- [ ] タッチスクリーン操作が可能（該当する場合）

### グラフィックス
- [ ] 描画が正常に表示される
- [ ] DXVK（DirectX → Vulkan）が機能する
- [ ] 解像度設定が反映される

## インストール方法

### 方法1: 手動配置
1. Windows PCでゲームをダウンロード
2. USB経由でAndroidデバイスに転送
3. アプリの「Import」機能で追加

### 方法2: Steam（該当する場合）
1. Steamアカウントと連携
2. ライブラリから無料ゲームをインストール

### 方法3: Winlatorコンテナ内でダウンロード
1. Winlatorコンテナを起動
2. Wine内でブラウザを開く（例: Wine Internet Explorer）
3. ゲームをダウンロードしてインストール

## トラブルシューティング

### ゲームが起動しない
- Wine prefixが初期化されているか確認
- エラーログを確認（Logcat経由）
- 必要なランタイム（DirectX, .NET Framework）がインストールされているか確認

### パフォーマンスが悪い
- Box64のdynarecが有効になっているか確認
- DXVKが有効になっているか確認
- ゲームの設定で低解像度・低設定を試す

### コントローラーが動作しない
- Android設定でコントローラーが認識されているか確認
- ゲーム内の設定で入力デバイスを確認
- InputDevice APIのログを確認
